\section{Scale heights}
\subsection{ Scale-height at ground-level and 100km altitude}
Task: calculate scale height at height = 0km and =100km for all species added together, calculated from the file MSIS.dat. 

The equation for scaleheight: 
\begin{equation}
H = \frac{ k_{\text{B}}T}{ mg}
\end{equation}
I assumed that a standard for gravitational acceleration at earth is g0 = 9.82 $\frac{m}{s^2}$. 
A formula for the variation of g is
\begin{equation}
g(z) = g(0) \frac{R^2_{\text{E}}}{(R_{\text{E}}+z)^2}
\end{equation}

First I imported necessary libraries and defined constants.
\begin{python}
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

data = pd.read_csv("progrTasks/1atmosphere/MSIS.dat",sep=r"\s+", skiprows=16)

##constants
k_B = 1.38e-23     #[m^2*kg/(K*s^2)] boltzmanns constant
g0 = 9.82 #[m/s] gravitational acceleration
R_E = 6378e3 #[m] earth radius
p0 = 101325 #[Pa] std pressure at sealevel
#O weighs 16amu or g/mol
#N2 weighs 28amu or g/mol
#O2 weighs 32amu or g/mol
\end{python}

Then I defined a function to calculate the scale height from the ground up to 600km, and using the average mass of the particles at each height (step 1km).
(The function can be used for constant g or g varying with height, for use in a later question).
\begin{python}
#calculate scaleheight --> H= k_B*T/(m*g)
#scale height for all species
def calc_Hall(data_file, height, g):
     #nr density all molecules
    n = data_file.iloc[height,1]+ data_file.iloc[height,2]+ data_file.iloc[height,3]
    #average molecule weight
    m = (data_file.iloc[height,1]*16 + data_file.iloc[height,2]*28 + data_file.iloc[height,3]*32)/n *1.66054 * 10e-28 #[kg]
    #constant g or z-dependent g
    if g =="g_z":
        g = g0 * R_E**2 /(R_E + height*1e3)**2
    else:
        g = g0
    #scale height
    H = (k_B * data_file.iloc[height,5]) / (m * g)
    return H
\end{python}
This gives for z = 0km: 7.689km and for z=10km: 6.287km

\subsection{pressure variations}
The task is to calculate atmospheric pressure variation from the following equation:
\begin{equation}
\frac{\partial p}{\partial z} = - \frac{m(z)g(z)}{k_{\text{B}}T(z)}p(z)
\end{equation}
First by numerical integration and then by solving it (partly) analytically. 

Numerical integration: the following function outputs an array with the pressure for all heights from 0 to 600km.
(here dz = 1 because I already converted $H_{all}g_z $ into km when I stored the values for the scaleheight in the list!) 
\begin{python}
p_num = [0 for _ in range(len(H_all))]
p_num[0] = p0
dz = 1
for i in range(1,len(H_all)):
    p_num[i] = p_num[i-1] - (p_num[i-1]/H_all_gz[i-1])*(dz)
p_num = np.array(p_num)
\end{python}
Analytical solution:
We need to integrate 
\begin{equation}
\int_{p_{0}}^{p} \frac{1}{p} dp  = -  \int_{z_{0}}^{z} \frac{k_{\text{B}}T(z)}{m(z)g(z)} dz 
\end{equation}
This gives the equation 
\begin{equation}
p = p_{0}exp(- \int_{z_{0}}^{z} \frac{k_{\text{B}}T(z)}{m(z)g(z)} dz )
\end{equation}
or as I already calculated the scaleheight earlier and want to reuse the results, in terms of scale height:
\begin{equation}
p = p_{0}exp(- \int_{z_{0}}^{z} \frac{1}{H(z)} dz )
\end{equation}
(also here dz = 1 for the same reason as above)
\begin{python}
H_num = [0 for _ in range(len(H_all))]
H_num[0] = 1/H_all_gz[0]
for i in range(1, len(H_all)):
    H_num[i] = H_num[i-1] + 1/(H_all_gz[i-1])*dz
